Online Service for real estate valuation

The SteppeResidence project is designed to predict the actual price of real estate in Kazakhstan. Supervised machine learning methods, including regression analysis, can help in solving this problem. Models based on it, based on the analysis of the parameters of apartments, will predict the most realistic price for real estate. Based on a database with 46 features and more than 40 thousand objects, machine learning models were built and the best model (XGBoost) was selected as the main one. According to the parameters, median absolute error is 4.67%, the mean absolute error is 6.81% and R² = 0.8. In the future, the quality of model predictions will improve with an increase in the number of records. All of these models were connected via Django to the website.
